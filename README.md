#### 使用说明

1. 自己开容器，环境选择python或者go都可以，安装时会让选择环境参数，1代表go，2代表python

2. 另一个需要设置参数是v2ray伪装名称，自行设置（尽可能选择英文字母）

3. 其他使用教程照搬IBMYes即可，直达链接：https://github.com/CCChieh/IBMYes

4. 能否使用一切随缘，有问题不要发issue，本代码仅自用，顺带方便一下mjj
